- **Animation** — Framer Motion, Rough Notation
- **Analytics** — Vercel Analytics

## 📦 Installation

1. Clone the repo:
    ```bash
    git clone https://github.com/ayuugoyal/portfolio.git
    cd portfolio
    ```

2. Install dependencies:
    ```bash
    pnpm install
    ```

3. Run dev server:
    ```bash
    pnpm dev
    ```

4. Open [http://localhost:3000](http://localhost:3000)
